# SoundPad server

# Порядок работы: 
1) Репозиторий копируем на компьютер
2) Внутри репозитория можно добавлять/удалять папки со звуками
3) Для обновления сервера Запускаем батник и он генерирует Json с новой структурой папок и сразу пушит изменения на сервер
4) После обновления сервера Приложение при новом запуске забирает этот JSON с сервера, получает из него новую структуру и прямые ссылки на все файлы и скачивает их.
